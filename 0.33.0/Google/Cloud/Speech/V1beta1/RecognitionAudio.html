RecognitionAudio

class RecognitionAudio extends Message

Contains audio data in the encoding specified in the RecognitionConfig.

Either content or uri must be supplied. Supplying both or neither returns [google.rpc.Code.INVALID_ARGUMENT][google.rpc.Code.INVALID_ARGUMENT]. See audio limits.
