DataTransferServiceGrpcClient

class DataTransferServiceGrpcClient extends BaseStub

The Google BigQuery Data Transfer Service API enables BigQuery users to configure the transfer of their data from other Google Products into BigQuery.

This service contains methods that are end user exposed. It backs up the frontend.

CheckValidCreds(CheckValidCredsRequest $argument, array $metadata = [], array $options = [])

Returns true if valid credentials exist for the given data source and requesting user.

Some data sources doesn't support service account, so we need to talk to them on behalf of the end user. This API just checks whether we have OAuth token for the particular user, which is a pre-requisite before user can create a transfer config.
